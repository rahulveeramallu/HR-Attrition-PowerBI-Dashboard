# HR Attrition Dashboard - Power BI

## Project Overview

This project analyzes employee attrition using IBM HR Analytics dataset.
The dashboard provides insights into employee turnover and key factors affecting attrition.

## Tools Used

* Power BI
* Python (Colab)
* Dataset: IBM HR Analytics

## Dashboard Features

* Attrition by Department
* Attrition by Age Group
* Attrition by Gender
* Attrition by Job Role
* Attrition by Overtime
* Attrition Rate KPI
